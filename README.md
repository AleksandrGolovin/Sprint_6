# Sprint_6
Автоматизация тестирования сервиса "Самокат" с использованием Python, pytest и Selenium WebDriver

## Тестовые классы

### 1. TestMainPage (Главная страница)
**Файл:** `tests/test_main_page.py`

**test_questions_and_answers**  
Проверка блока "Вопросы-ответы":  
- Переход на главную страницу  
- Подтверждение cookie  
- Проверка 8 вопросов (параметризованный тест)  
- Сравнение текста вопроса и ответа  

---

### 2. TestOrderPage (Страница заказа)
**Файл:** `tests/test_order_page.py`

**test_create_order**  
Проверка оформления заказа:  
- Оформление через верхнюю и нижнюю кнопки  
- Заполнение личных данных  
- Выбор параметров аренды  
- Проверка успешного оформления  

---

### 3. TestTransitionPage (Переходы)
**Файл:** `tests/test_transition_page.py`

**test_transition_via_yandex_logo**  
Проверка перехода на Дзен:  
- Клик на логотип Яндекса  
- Переключение на новую вкладку  
- Проверка страницы Дзен  

**test_transition_via_scooter_logo**  
Проверка перехода на главную:  
- Клик на логотип Самоката  
- Проверка главной страницы  

---

## Фикстуры
- **chrome_driver**: Chrome в инкогнито (1400x800)
- **firefox_driver**: Firefox в приватном режиме (1400x800)

## Тестовые данные
`data.py` содержит:
- Вопросы-ответы (QNA_DATA)
- Данные для заказов (ORDER_DATA_1, ORDER_DATA_2)
- URL-адреса страниц (Urls)
